#
# 概要
#

OpenGL, GLUT, glpngを使って作られたグラフィカルなじんちとりゲームです．
あいてと交互に線を引き，四角を囲うと得点を得ることができます．
最終的に点数が大きい方が勝ちです．


#
# ビルド方法
#

ビルドのためには以下が必要です．
Mac
    Xcodeもしくは，Developer Tools
    glpng
Windows
    フルインストールされたCygwin
    glpng
glpngについては
http://teacher.nagano-nct.ac.jp/ito/Springs_of_C/index.html
よりライブラリをダウンロードできます．

以下のコマンドでビルドできます．
Windowsの場合はCygwinで，Macの場合は標準インストールされている端末から実行してください．
$はプロンプト記号を示しているため，それ以降を入力してください．
Windowsの場合はアイコンが付きます．
-----------------
$ make
-----------------


#
# 実行方法
#

makeコマンドによって生成されたj13411.exeを実行します．
Macの場合は拡張子なしのファイルを実行します．
-----------------
$ ./j13411.exe
-----------------


#
# ルール
#

相手と交互にクリックして線を引きます．
四角を囲えば，得点を得ることができます．
得点が高い方の勝利です．


#
# 操作方法
#

タイトル画面よりスペースキーを押すことでゲームのプレイができます．
ゲームが終了すると結果画面に行きます．
結果画面でスペースキーを押すことでタイトルに戻れます．
ゲームプレイ中に線を引くためには線をクリックをします．
